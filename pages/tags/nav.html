Tag <nav>

Definição e uso

A tag <nav> define um conjunto de links de navegação.

Observe que nem todos os links de um documento devem estar dentro de um elemento nav. O elemento nav destina-se apenas aos principais blocos de links de navegação.

Navegadores, como leitores de tela para usuários desativados, podem usar esse elemento para determinar se devem omitir a renderização inicial desse conteúdo.

Exemplo

<nav>

a href="/html/">HTML</a>
a href="/css/">CSS</a>
a href="/js/">JavaScript</a>
a href="/python">Python</a>

</nav>
