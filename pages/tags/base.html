Tag <base>

Definição e uso

A tag <base> especifica a URL base e/ou destino para todas as URLs relativas em um documento.
A tag <base> deve ter um href ou um atributo de destino presente, ou ambos.
Só pode haver um único elemento <base> em um documento e ele deve estar dentro do elemento <head>.

Exemplo

<head>

<base href="../"> target="_blank">

</head>


<body>

<img src="../image/logo/logotce.png">
<a href="span.html">Página da tag span</a>

</body>

Atributos

Atributo Value Descrição
href URL Especifica o URL base para todos os URLs relativos na página
target _blank
_parent
_self
_top
off		 Especifica o destino padrão para todos os hiperlinks e formulários na página
