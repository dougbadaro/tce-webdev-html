Tag <optgroup>

Definição e uso

A tag <optgroup> é usada para agrupar opções relacionadas em um elemento <select> (lista suspensa).

Se você tiver uma longa lista de opções, grupos de opções relacionadas são mais fáceis de manipular para um usuário

Exemplo

<label for="cars">Choose a car:<label for="cars">

<select name="cars" id="cars">

<optgroup label="Swedish Cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
</optgroup>

</select>

Atributos

Atributo Value Descrição
disabled disable Especifica que um grupo de opções deve ser desabilitado
label text O elemento HTML <optgroup> especifica um rótulo para um grupo com opções
