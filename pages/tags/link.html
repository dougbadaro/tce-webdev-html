Tag <link>

Definição e uso

A tag <link> define o relacionamento entre o documento atual e um recurso externo.

A tag <link> é usada com mais frequência para vincular a folhas de estilo externas ou para adicionar um favicon ao seu site.

O elemento <link> é um elemento vazio, contém apenas atributos.

Exemplo

<head>

<link rel="stylesheet" href="styles.css">

</head>
