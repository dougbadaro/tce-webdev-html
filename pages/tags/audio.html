Tag <audio>

Definição e uso

A tag <audio> em HTML é usada para incorporar e reproduzir conteúdo de áudio em uma página web. Essa tag permite que você insira arquivos de áudio, como música ou sons, diretamente no seu documento HTML, tornando-os reproduzíveis no navegador.

Exemplo

<audio controls>

<source src="horse.ogg"> type="audio/ogg">

</audio>


Atributos

Atributo Value Descrição
autoplay autoplay Especifica que o áudio começará a tocar assim que estiver pronto
controls controls Especifica que os controles de áudio devem ser exibidos (como um botão de reprodução/pausa, etc.)
loop loop Specifies that the audio will start over again, every time it is finished
muted muted Especifica que a saída de áudio deve ser silenciada
preload auto
metadata
none		 Especifica se e como o autor acha que o áudio deve ser carregado quando a página for carregada
src URL Especifica a URL do arquivo de áudio
