Tag <track>

Definição e uso

A tag <track> especifica trilhas de texto para elementos <audio> ou <video>. Este elemento é usado para especificar legendas, arquivos de legenda ou outros arquivos contendo texto, que devem estar visíveis quando a mídia estiver sendo reproduzida. As trilhas são formatadas no formato WebVTT (arquivos .vtt).

Exemplo

<video width="320" height="240" controls>
<source src="forrest_gump.mp4" type="video/mp4">
<source src="forrest_gump.ogg" type="video/ogg">
<track src="fgsubtitles_en.vtt"kind="subtitles" srclang="en" label="English">
<track src="fgsubtitles_no.vtt"kind="subtitles" srclang="no" label="Norwegian">

Atributos

Atributo Value Descrição
default default Especifica que a faixa deve ser habilitada se as preferências do usuário não indicarem que outra faixa seria mais apropriada
kind captions
chapters
descriptions
metadata
subtitles		 Especifica o tipo de faixa de texto
label text Especifica o título da faixa de texto
src URL Obrigatório. Especifica a URL do arquivo de trilha
srclang language_code Especifica o idioma dos dados do texto da faixa (obrigatório se kind="legendas")
