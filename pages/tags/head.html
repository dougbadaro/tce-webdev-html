Tag <head>

Definição e uso

A tag <head> é um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a tag <body>

Metadados são dados sobre o documento HTML. Os metadados não são exibidos.

Um elemento <head> pode conter:

  • <title>
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>

Exemplo

<DOCTYPE html>

<html lang="pt-BR">

<head>

<title>Título legal!</title>

</head>

<body>

<h1>Título muito legal!</h1>

<p>Parágrafo legal!</p>

</body>

</html>
