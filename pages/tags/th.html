Tag <th>

Definição e uso

A tag <th> define uma célula de cabeçalho em uma tabela HTML.

Uma tabela HTML tem dois tipos de células:

  • Células de cabeçalho - contém informações de cabeçalho (criadas com o elemento <th>)
  • Células de dados - contém dados (criados com o elemento <td>)

O texto nos elementos <th> está em negrito e centralizado por padrão.

O texto nos elementos <td> são regulares e alinhados à esquerda por padrão.

Exemplo

<table style="width:100%">

<tr>
<th style="text-align:left">Mês</th>
 <th style="text-align:left">Poupança</th>
</tr>

</table>

colspan number Especifica o número de colunas que uma célula de cabeçalho deve abranger
headers header_id
off		 Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada
rowspan number Especifica o número de linhas que uma célula de cabeçalho deve abranger
scope col
colgroup
row
rowgroup		 Especifica se uma célula de cabeçalho é um cabeçalho para uma coluna, linha ou grupo de colunas ou linhas
