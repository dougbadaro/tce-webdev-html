Tag <li>

Definição e uso

A tag <li> define um item de lista.

A tag <li> é usada dentro de listas ordenadas (<ol>), listas não ordenadas (<ul>) e em listas de menu (<menu>).

Em <ul> e <menu>, os itens da lista geralmente serão exibidos com marcadores.

Em <ol>, os itens da lista geralmente serão exibidos com números ou letras.

Exemplo

<ol>
 <li>Café</li>
<li>Chá</li>
<li>Leite</li>
</ol>

<ul>
 <li>Café</li>
<li>Chá</li>
<li>Leite</li>
</ul>

Atributos

Atributo Value Descrição
value number Somente para listas <ol>. Especifica o valor inicial de um item de lista. Os itens da lista a seguir serão incrementados a partir desse número
