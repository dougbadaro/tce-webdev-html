Tag <textarea>

Definição e uso

A tag <textarea> define um controle de entrada de texto de várias linhas.

O elemento <textarea> geralmente é usado em um formulário para coletar entradas do usuário, como comentários ou revisões.

Uma área de texto pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa.

O tamanho de uma área de texto é especificado pelos atributos cols e rows(ou com CSS).

O atributo name é necessário para fazer referência aos dados do formulário após o envio do formulário (se você omitir o atributo name, nenhum dado da área de texto será enviado).

O atributo id é necessário para associar a área de texto a um rótulo.

Exemplo

<label for="w3revisao"Revisão W3Schools</label>


<textarea id="w3revisao" name="w3revisao" rows="4" cols="50"
Em w3schools.com você aprenderá como fazer um site. Eles oferecem tutoriais gratuitos em todas as tecnologias de desenvolvimento web.

</textarea>

Atributos

Atributo Value Descrição
autofocus autofocus Especifica que uma área de texto deve obter foco automaticamente quando a página for carregada
cols action Especifica a largura visível de uma área de texto
dirname textareaname.dir Especifica que a direção do texto da área de texto será enviada
disabled disabled Especifica que uma área de texto deve ser desativada
form form.id Especifica a qual formulário a área de texto pertence
maxlength number Especifica o número máximo de caracteres permitidos na área de texto
name text Especifica um nome para uma área de texto
placeholder text Especifica uma dica curta que descreve o valor esperado de uma área de texto
readonly readonly Especifica que uma área de texto deve ser somente leitura
required required Especifica que uma área de texto é obrigatória ser preenchida
rows number Especifica o número visível de linhas em uma área de texto
wrap hard
soft		 Especifica como o texto em uma área de texto deve ser quebrado quando enviado em um formulário
