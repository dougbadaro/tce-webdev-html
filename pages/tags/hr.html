Tag <hr>

Definição e uso

A tag <hr> define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).

O elemento <hr> geralmente é exibido como uma régua horizontal usada para separar o conteúdo (ou definir uma alteração) em uma página HTML.

Exemplo

<h1>Os principais idiomas da Web</h1>

<p>
HTML é a linguagem de marcação padrão para criar páginas da Web. HTML descreve a estrutura de uma página da Web e consiste em uma série de elementos. Os elementos HTML dizem ao navegador como exibir o conteúdo.
