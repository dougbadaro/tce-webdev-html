Tag <hr>

Definição e uso

A tag <hr> define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).

O elemento <hr> geralmente é exibido como uma régua horizontal usada para separar o conteúdo (ou definir uma alteração) em uma página HTML.

Exemplo

<h1>The Main Languages of the Web</h1>

<p>HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content.</p>
