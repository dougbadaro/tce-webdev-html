Tag <source>

Definição e uso

A tag <source> é usada para especificar recursos de mídia, como áudio, vídeo ou imagens, que serão incorporados em uma página da web. A tag <source> é geralmente usada em conjunto com a tag <video> ou <audio>, dentro do qual é colocado como um elemento filho.

Exemplo

<audio controls>

<source src="caminho/para/arquivo-de-audio.mp3" type="audio/mpeg">

<source src="caminho/para/arquivo-de-audio.ogg" type="audio/ogg">

</audio controls>

Atributos

Atributo Value Descrição
media media_query Aceita qualquer consulta de mídia válida que normalmente seria definida em um CSS
sizes Especifica tamanhos de imagem para diferentes layouts de página
src URL Obrigatório quando <source> é usado em <audio> e <video>. Especifica a URL do arquivo de mídia
srcset URL Obrigatório quando <source> é usado em <picture>. Especifica a URL da imagem para usar em diferentes situações
type MIME-type Especifica o tipo MIME do recurso
