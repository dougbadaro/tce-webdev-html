Tag <span>

Definição e uso

A tag <span> é um contêiner embutido usado para marcar uma parte de um texto ou uma parte de um documento.
A tag <span> é facilmente estilizada por CSS ou manipulada com JavaScript usando o atributo class ou id.
A tag <span> é muito parecida com o elemento <div>, mas <div> é um elemento de nível de bloco e <span> é um elemento inline.

Exemplo

<p>My mother has <span style="color: blue">olhos azuis</span>
