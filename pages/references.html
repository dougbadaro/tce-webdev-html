Referências

Consulte a suportabilidade da tag de sua escolha em diferentes navegadores.

Tag Descrição
<!DOCTYPE> Todos os documentos HTML devem começar com uma declaração <!DOCTYPE>.
<abbr> A tag <abbr> define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
<address> A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo.
<audio> A tag <audio> do HTML5 permite adicionar arquivos de áudio em uma página da web, sem a necessidade de plugins ou software externo.
<b> A tag <b> especifica texto em negrito sem nenhuma importância extra.
<base> A tag <base> especifica a URL base e/ou destino para todas as URLs relativas em um documento.
<body> A tag <body> define o corpo do documento.
<br> A tag <br> insere uma única quebra de linha.
<button> A tag <button> define um botão clicável.
<dfn> A tag <dfn> representa o "elemento de definição" e especifica um termo que será definido dentro do conteúdo.
<dialog> A tag <dialog> define uma caixa de diálogo ou subjanela.
<em> A tag <em> é usada para definir o texto enfatizado. O conteúdo interno geralmente é exibido em itálico.
<figcaption> A tag <figcaption> define uma legenda para um elemento <figure>.
<footer> A tag <footer> define um rodapé para um documento ou seção.
<form> A tag <form> é usada para criar um formulário HTML para entrada do usuário.
<head> O elemento <head> é um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a tag <body>.
<hr> A tag <hr> define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).
<kbd> A tag <kbd> é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
<li> A tag <li> define um item de lista.
<link> A tag <link> define o relacionamento entre o documento atual e um recurso externo.
<nav> A tag <nav> define um conjunto de links de navegação.
<optgroup> A tag <optgroup> é usada para agrupar opções relacionadas em um elemento <select> (lista suspensa).
<q> A tag <q> define uma citação curta.
<rt> A tag <rt> define uma explicação ou pronúncia de caracteres (para tipografia do Leste Asiático) em uma anotação ruby.
<section> A tag <section> define uma seção em um documento.
<span> A tag <span> é um contêiner embutido usado para marcar uma parte de um texto ou uma parte de um documento.
<source> A tag <source> é usada para especificar vários recursos de mídia para elementos de mídia, como <video>, <audio> e <picture>.
<strong> A tag <strong> é usada para definir texto com forte importância.
<sup> A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW[1].
<td> A tag <td> define uma célula de dados padrão em uma tabela HTML.
<textarea> A tag <textarea> define um controle de entrada de texto de várias linhas.
<th> A tag <th> define uma célula de cabeçalho em uma tabela HTML.
<track> A tag <track> especifica trilhas de texto para elementos <audio> ou <video>.
